# History Cleaner

by GhostKnight340

Browser History and Bookmark Cleanup Utility for Windows.

## Download

Download the latest version from the Releases section.

## Features

- Browser history scanning
- Bookmark scanning
- Bookmark Grid View
- History cleanup
- Bookmark cleanup
- Automatic backups
- Restore support
- CSV exports
- TXT imports for domains and keywords

## Supported Browsers

- Opera
- Google Chrome
- Microsoft Edge
- Brave

## Notes

- The application ships with an empty configuration.
- Users must add their own domains and keywords.
- Windows may display a warning because the application is unsigned.
